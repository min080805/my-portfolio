# To-Do List Web App

## Description
This is a simple, interactive to-do list web application built with HTML, CSS, and JavaScript. Users can add tasks, mark them as completed, and remove them.

## Features
- Add tasks using an input field
- Click to mark tasks as completed (strikethrough)
- Remove tasks with a delete button
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open the `index.html` file in your web browser. You can add tasks, and they will appear in the list below. Clicking a task marks it complete; clicking the delete button removes it.
